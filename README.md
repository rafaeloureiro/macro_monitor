# Macro Monitor

Macro Monitor is a macroeconomic monitoring panel designed to track, organize,
and visualize key economic indicators for analytical and decision-making
purposes.

## Overview

The project focuses on centralizing reliable macroeconomic data and presenting
it in a clear and structured way. It aims to support economic analysis by
providing historical series and indicators commonly used in macroeconomic and
financial assessments.

## Covered Indicators

- Interest rates
- Inflation
- Real interest rates
- Implicit interest rates
- Gross Domestic Product (GDP)

## Data Sources

The project relies on official and reputable data providers, including:

- Central Bank of Brazil (BACEN)
- Brazilian Institute of Geography and Statistics (IBGE)
- Institute for Applied Economic Research (IPEA)

## Project Goals

- Centralize macroeconomic indicators in a single source
- Enable historical analysis and comparisons
- Support the development of dashboards and visual tools
- Provide a clean and extensible Python codebase
